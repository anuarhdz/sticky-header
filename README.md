> Sticky header pattern example

## Dependencies
* ParcelJS
* TailwindCSS
* PostCSS

## Install

```sh
npm install
```

## Usage

```sh
npm run start
```

## Build

```sh
npm run build
```

## HTML Markup 
```html
<header id="main-header" class="main-header">
  <sticky-header>
    <div class="components-container">
      <div id="top-header">
        ...
      </div>
      <div id="bottom-header">
        ...
      </div>
    </div>
  </sticky-header>
</header>
```
